# TopAces - Military Aircraft Comparison Simulator

Web application for comparing and simulating combat capabilities of military aircraft.

## Features
- 94+ military aircraft database
- Combat simulator with advanced algorithm
- User authentication with email verification
- Admin panel for database management
- Combat history tracking

## Tech Stack
- PHP 8
- MySQL
- HTML5, CSS3, JavaScript
- PHPMailer
